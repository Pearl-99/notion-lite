# Notion Lite 📝

A modern note-taking web application inspired by Notion, built using React and Vite.

## 🚀 Features

* Block-based editor (like Notion)
* Modular components architecture
* Drag & drop blocks
* Light & Dark theme
* Fast performance using Vite
* Local storage support (data persistence)

## 🛠️ Tech Stack

* React
* Vite
* JavaScript
* Tailwind CSS
* LocalStorage

## 📁 Project Structure

* `components/` → UI components (editor, layout)
* `hooks/` → Custom React hooks

## ▶️ Run Locally

```bash
npm install
npm run dev
```
